-De kécé? Poutines Guide?

Poutines Guide est un projet web de Théophile Nzungize, un jeune étudiant en développement web de la ville de Québec.
Il s'agit d'un répertoire des différents restaurants de poutines du Québec et de leurs différents menus.
Poutines Guide se veut d'être une célébration du plat national du Québec au travers de ses variétés et l'originalité de tous ceux qui participent à l'essor de ce plat dans le monde.

Okay. Si les termes techniques vous interessent, le site web roule sur une plateforme que l'on appelle Node.js.

-Node jséquoi?

Oui. Node.js. Une plateforme ultra rapide qui me permet de créer un site complet en Javascript. De A à Z. En fait le site web utilise 4 composantes essentielles qu'on connaît sous le nom du MEAN Stack.
Simplement:
MongoDB, la base de données qui me permet de gérer les restos, les poutines et le contenu en général (avec un coup de main de Mongoose, un module de connexion),
Express.js, le module qui est en quelque sorte le serveur pour communiquer entre les différents morceaux du site web,
Angular.js, un framework Javascript puissant crée par Google pour afficher tout ce que vous voyez,
Nodejs, la plateforme complète qui rend le tout possible!
Voila.
